# homebrew-tap

Homebrew formulae for [@RensHoogendam](https://github.com/RensHoogendam)'s tools.

## Usage

```zsh
brew install RensHoogendam/tap/<formula>
```

## Available formulae

| Formula | Description |
| --- | --- |
| [`zsh-bw-keys`](https://github.com/RensHoogendam/zsh-bw-keys) | Lazy Bitwarden secret injection for zsh environment variables |
